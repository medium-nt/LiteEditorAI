# LiteEditor — module-kit

Набор для создания пользовательских модулей LiteEditor: панелей, открывающихся в правом
слоте редактора (как встроенные Git, Базы данных, Контейнеры).

- **`GUIDE.md`** — полная спецификация: манифест, API `ctx` v1, правила оформления, пример.
  Самодостаточна — написана так, чтобы её целиком можно было отдать ИИ-агенту.
- **`examples/calculator/`** — рабочий модуль-образец (тот же код, что в GUIDE).
- **`ai/`** — заготовки контекста для агентов (CLAUDE.md для Claude Code, AGENTS.md для Codex);
  редактор кладёт их в папку модуля при создании через мастер.

## Быстрый старт

1. В редакторе: меню **«Модули → Создать модуль…»** — мастер создаст заготовку (задание
   ляжет в `TASK.md`) и откроет терминал модуля. Агента запускаете сами — любого (claude,
   codex, …) и со своими настройками: инструкции он подхватит из `CLAUDE.md`/`AGENTS.md`.
   Это основной путь.
2. Вручную: скопируйте `examples/calculator/` в `~/.LiteEditorAI/modules/calculator/`
   (Windows: `C:\Users\<имя>\.LiteEditorAI\modules\`), затем «Модули → Пересканировать».

Модуль исполняется в окне редактора без песочницы: установка модуля = доверие его коду,
как установка любой программы. Поле `capabilities` манифеста — честная декларация
возможностей, показываемая пользователю.
