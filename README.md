## Зачем

Когда код всё чаще пишет агент (Claude Code, Codex, Qwen, Kimi…), а не ты сам, привычный редактор
встаёт с ног на голову: в центре уже не файл, а **разговор с агентом в терминале**. Тяжёлый IDE с
сотней панелей под это избыточен, а голый терминал неудобен — не видно, какой агент уже закончил, какой
ждёт твоего ответа, что он наменял в файлах и где вообще какой проект.

**LiteEditorAI** построен вокруг этого: главный здесь твой терминал, а просмотр кода, дерево и git живут
рядом ровно настолько, насколько нужно следить за агентом, — и прячутся одной кнопкой, когда не нужны.
Это нарочно лёгкий и тихий инструмент: открыл папку — и сразу за дело, без долгой настройки.

## Что умеет

- 🛰 **Удалённый пульт (Android) — управляй редактором с планшета или телефона.** Подключение по аккаунту
  через self-hosted релей: **полноценный терминал** со своей экранной клавиатурой, переключение
  проектов/вкладок, просмотр и скачивание файлов с ПК, безопасный перезапуск редактора. Безопасность
  всерьёз: **одобрение устройства на ПК** — даже зная пароль, чужое устройство не получит доступ к
  терминалу; плюс защита от перебора пароля, отзывные сессии и отзыв устройств при потере.
  [Как подключить →](#удалённый-пульт-android)
- 🖥 **Несколько терминалов на проект — вкладки.** У каждого проекта свои живые shell-вкладки: запусти
  агента в одной, dev-сервер в другой, разовую команду в третьей. Вкладки переименовываются, а их **имена
  сохраняются между перезапусками**.
- 🚦 **Видно состояние агента с одного взгляда:** работает (спиннер) · ждёт твоего ответа (янтарный) ·
  готов (зелёный). Плюс системные уведомления и счётчик «сколько агентов ждут» — не нужно щёлкать каждый,
  чтобы понять, кто освободился. Индикатор на карточке — агрегат по всем вкладкам проекта.
- 📝 **Заметки-промпты и авто-очередь:** копишь следующие промпты карточками, а очередь отправляет их в
  терминал по одному — после каждого хода агента ты решаешь, слать ли следующий.
- 💬 **Чат с моделями через OpenRouter (свой ключ):** добавь свой API-ключ — рядом с проектами появится
  плашка-чат. Выбор любой модели из OpenRouter с поиском, ценой за 1M токенов и размером контекста; ответы
  **стримом** с подсветкой кода и разметкой Markdown; **несколько сессий на ключ** и настраиваемое окно
  контекста (сколько последних сообщений уходит модели). Сгенерированные изображения показываются прямо в
  чате — клик увеличивает, есть кнопка «Скачать». На плашке видно **израсходовано / лимит** ключа с кнопкой
  обновления. Для быстрого вопроса модели, когда не нужен полноценный агент в терминале. Ключи хранятся локально.
- 👁 **Код рядом с терминалом:** дерево файлов, подсветка, правка и сохранение, поиск, дифф изменений vs
  HEAD, превью Markdown / картинок / HTML. Дерево само обновляется, пока агент правит файлы.
- ⎇ **Git под рукой:** статус прямо в дереве, ветки, commit / push / pull / fetch, откат файла — не выходя
  из окна.
- 🗂 **Порядок в проектах:** категории (создать, переименовать, менять порядок, свернуть; плюс «Архив» для
  отложенных), избранное, акцент-цвета, авто-скан папки с проектами.
- 🎨 **6 тем оформления:** Неоморфизм (по умолчанию), Стекло, Material, Catppuccin, Gruvbox, Aurora —
  тёмные, аккуратные, с единым акцентом. Терминал перекрашивается под тему.
- 🔔 **Уведомление об обновлениях:** редактор сам сверяется с последним релизом и подсвечивает плашку у
  номера версии, когда выходит новая — клик ведёт на страницу загрузки (само ничего не ставит).
- 🪶 **Без перегруза и кросс-платформенно:** ничего не настраивать на старте, маленькая панель настроек,
  Ctrl+C/Ctrl+V в терминале работают в любой раскладке на Linux и Windows.

## Удалённый пульт (Android)

Управляй редактором с планшета или телефона через защищённый релей — терминал, проекты и файлы ПК в руке.

1. Скачай **`liteeditor-pult-*.apk`** со страницы [Releases](https://github.com/DanielLetto2020/LiteEditorAI/releases)
   и установи на Android (разреши установку из неизвестных источников).
2. В редакторе на ПК: меню **«Пульт»** → зарегистрируй аккаунт (логин/пароль).
3. В приложении на устройстве войди тем же аккаунтом.
4. Меню → **«Подключить это устройство»** → одобри на ПК (сверь код). Готово.

**Безопасность.** Знание пароля само по себе не даёт доступ к терминалу — устройство нужно **одобрить на
ПК**. Есть защита от перебора пароля, отзывные сессии и кнопка **«Выйти на всех устройствах»** на случай
потери планшета.

> Связь идёт через релей, который **видит трафик** (не end-to-end) — для приватного кода учитывай это.
> Пульт в стадии **alpha**.

## Установка

Готовые сборки — на странице [**Releases**](https://github.com/DanielLetto2020/LiteEditorAI/releases).

### Ubuntu / Debian (x64)
```bash
sudo apt install ./LiteEditorAI_*.deb
```
Одна команда — поставит приложение и подтянет зависимости. Запуск — иконка **LiteEditorAI** в меню приложений.

### Windows (x64)
Скачай **portable**-архив `LiteEditorAI_*-win.zip`, распакуй в любую папку и запусти **`LiteEditorAI.exe`**.
Установка не нужна. Приложение пока без цифровой подписи — SmartScreen может предупредить:
«Подробнее» → «Выполнить в любом случае».

## Сборка из исходников

```bash
npm install        # зависимости + сборка node-pty под Electron
npm start          # сборка фронта + запуск
```

Требуется Node.js 18+ (Linux/Windows x64). Для разработчиков — [CONTRIBUTING.md](CONTRIBUTING.md).

## Горячие клавиши

| Клавиши | Действие |
|---|---|
| `Ctrl+Shift+T` / `Ctrl+Shift+W` | новая / закрыть вкладку терминала |
| `Ctrl+PageUp` / `Ctrl+PageDown` | переключение вкладок терминала |
| `Ctrl+C` / `Ctrl+V` | копировать выделение / вставить (в любой раскладке) |
| `Ctrl+\` | режим «один терминал» |
| `Ctrl+K` | палитра команд |
| `Ctrl+F` | поиск (в терминале или в открытом файле) |
| `Ctrl+S` | сохранить файл |
| `Ctrl+1..9` / `Ctrl+Tab` | переключение проектов |
| `Ctrl + +/−` | размер шрифта · `F11` — полный экран |
