**LiteEditorAI v1.1.39** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Новый модуль «ИИ компания».** Команда ИИ-агентов работает над вашим проектом: агент-директор сам нанимает сотрудников, раздаёт задачи и доводит цель до результата — а вы наблюдаете за работой вживую.

## ✨ Добавлено
- **Модуль «ИИ компания»** (за активным проектом) — агент-**директор** на родном механизме сабагентов Claude:
  получает цель, разбивает её на задачи, нанимает и вызывает специалистов (кодер, ревьюер и другие), ведёт
  общую доску задач и доводит работу до конца.
- **Живой лог работы** с временными метками и **статусы сотрудников** (работает / готов) — видно, кого
  директор позвал и что делает команда.
- **Доска задач** с прогресс-баром (`.lite/company/board.md`) — общий журнал для вас и команды.
- **Штат компании** с редактированием ролей, **библиотека готовых ролей** (архитектор, тестировщик,
  документатор, безопасник, рефакторер, девопс) в один клик и **шаблоны целей**.
- **Режим «План»** (сухой прогон без правок на диске), **лимит бюджета**, **очередь целей** и
  **история прогонов** со стоимостью.
- **Память компании** (`.lite/company/notes.md`) — уроки между прогонами; **обзор изменений** (git diff) по итогам.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
