**LiteEditorAI v1.0.182** — лёгкий редактор для работы с ИИ-агентами в терминале.

Встроенный таск-трекер **IterFlow** теперь умеет не только показывать задачи, но и **изменять** их —
работайте над задачами прямо из редактора, не уходя в браузер. И у проекта появился официальный сайт.

## ✨ Добавлено

- **IterFlow — слой записи (CRUD + жизненный цикл).** Раньше модуль работал только на чтение, теперь:
  - **Итерации** — создание, переименование, дедлайн, удаление и переходы стадий (сдать / согласовать / принять / вернуть);
  - **Задачи** — создание, правка, удаление, отметка «готово», смена статуса на **Канбане**;
  - **Заметки проекта (туду)** — создание, правка и удаление автором.

  Все ограничения (стадия, роль, авторство, заморозка проекта) проверяет сервер IterFlow.
- **Сайт проекта — [lite-editor-ai.ru](https://lite-editor-ai.ru).**

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
