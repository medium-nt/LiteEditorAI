**LiteEditorAI v1.0.108** — лёгкий редактор для работы с ИИ-агентами в терминале.

«Заметки» проекта превратились в полноценный **TODO-список** со статусами, важностью и сортировкой.

## ✅ Задачи вместо заметок

Вместо карточек — компактные строки задач, удобно вести и отмечать прогресс:

- **Статус** (клик по чекбоксу слева циклит): ☐ к выполнению → ◐ в работе → ☑ выполнено. **Выполненные задачи не удаляются** — просто помечаются (зачёркнуты и приглушены). Реальное удаление — отдельной корзиной с подтверждением.
- **Важность** (флажок): обычная → важная → срочная, с цветной полосой слева.
- **Сортировка** (запоминается): вручную (перетаскивание / ▲▼), по важности, по статусу.
- Кнопки действий — иконками в стиле карточек проекта, всегда на виду. Отправка в терминал и **авто-очередь** работают как раньше; бейдж на карточке проекта считает активные (не выполненные) задачи.

> **Данные сохранены:** старые заметки открываются как задачи «к выполнению» — формат хранения совместим, ничего не теряется.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
