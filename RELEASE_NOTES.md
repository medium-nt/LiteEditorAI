**LiteEditorAI v1.1.28** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Вивер кода дорастает до полноценной IDE.** Git внутри окна переделан под PhpStorm, добавлены привычные по
VS Code / PhpStorm возможности редактора и уникальный «режим агента».

## ✨ Добавлено
- **Git под PhpStorm** в окне «Проект»: вкладка **«Коммит»** (цветовая кодировка статусов, чекбоксы выбора с
  tri-state групп, **Commit / Commit & Push**, stash + модалка списка stash), нижняя панель **«Ветки · История»**
  (коммиты слева → изменённые файлы коммита **деревом с иконками типов** справа, клик → дифф), выпадашка веток
  **Local / Remote** деревом с избранными и полным меню действий (checkout / merge / rebase / pull / сравнить /
  переименовать / удалить…).
- **Редактор как в IDE:** Find & Replace + regex (Ctrl+F), множественные курсоры и колоночное выделение,
  операции со строками (дубль/перенос/комментирование), **sticky scroll**, **git blame** строки, сворачивание
  `#region`, инлайн-превью цвета с пикером, подсветка **TODO/FIXME** + список, **split-превью** Markdown/HTML,
  **Zen-режим**, **колонка табов** с закреплением и контекстным меню, **автосохранение**.
- **Режим агента** (тумблер): ревью правок агента с откатом по ханкам, «спросить агента про выделение»
  (объяснить/отрефакторить/тест → в терминал), слой авторства строк (агент / вы).

## 🐛 Исправлено
- Заработали **минимап** и **прокрутка файла** (раскладка окна вивера).
- Гонки при быстром переключении файлов, ремап навигации/недавних при переименовании-перемещении-удалении,
  освобождение таймеров при закрытии окна, мелкие правки Git-панели и зачистка мёртвого кода.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
