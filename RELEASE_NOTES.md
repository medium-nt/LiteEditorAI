**LiteEditorAI v1.0.107** — лёгкий редактор для работы с ИИ-агентами в терминале.

Большой релиз: новый модуль для работы с удалёнными серверами и хостингами, миникарта кода в вивере и доводка интерфейса проектов.

## 🔌 Удалённые хосты — серверы и хостинги прямо в редакторе

Новый модуль в меню **«Модули»**. Профили подключений по **SSH / SFTP / FTP**, сгруппированные по категориям, с тестом соединения и входом **в один клик**.

- 🖥 **SSH — полноценный терминал на сервере.** Несколько живых сессий как **вкладки** с переключением. Вход **по паролю** или **по ключу из системы без пароля** (SSH-агент / `~/.ssh`, как привычный `ssh user@host`).
- 🫀 **Keepalive** с настраиваемым интервалом — чтобы соединение не рвалось при простое (как «галочка» в PhpStorm).
- 📁 **Просмотр файлов** у каждого подключения (кнопка-папка): дерево удалённых файлов по **SFTP** или **FTP/FTPS** с навигацией и чтением содержимого.
- 🔒 Пароли шифруются системным хранилищем ключей и не покидают бэкенд; режим «без пароля» ничего не хранит; FTPS проверяет сертификат.

## 🗺 Миникарта кода в вивере

Уменьшенная копия файла справа от кода (как в VSCode) — с рамкой текущей области и кликом-прыжком по файлу. Включается/выключается кнопкой в шапке вивера, состояние запоминается.

## 🧩 Доводка интерфейса

- **Пункт «Вивер»** в меню «Модули» — открыть просмотр файлов активного проекта.
- **Карточки проектов и категории — действия по наведению (hover-reveal):** кнопки спрятаны за свёрнутой стрелкой и выезжают при наведении — список спокойнее и компактнее.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
