**LiteEditorAI v1.0.180** — лёгкий редактор для работы с ИИ-агентами в терминале.

Срочный фикс упаковки поверх v1.0.179: установленное приложение не запускалось.

## 🐛 Исправлено

- **Приложение падало при запуске из дистрибутива** (`.deb` и `.zip`) с ошибкой
  `Error: Cannot find module './errledger'`. Модуль реестра ошибок (`errledger.js`) подключается напрямую,
  но не попадал в сборку. Теперь включён — установленное приложение снова стартует штатно.
  Затрагивало все сборки начиная с v1.0.167. Спасибо [@Eurgen](https://github.com/Eurgen) за репорт (#5).

> Если у вас v1.0.179 или раньше и приложение не открывалось — обновитесь до v1.0.180.

В остальном — содержимое v1.0.179: новый модуль **«Инструменты»** (локальный девтулз-комбайн: Base64 ↔
картинка, JSON-вьюер со сворачиванием, JSON/YAML, хэши, JWT, regex, diff, cron, цвет и др.).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
