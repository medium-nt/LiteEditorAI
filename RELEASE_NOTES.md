**LiteEditorAI v1.0.193** — лёгкий редактор для работы с ИИ-агентами в терминале.

Сквозная доработка панели **«Контейнеры»** (Docker / Podman): живое автообновление, аккуратные групповые
действия и заметно меньше фоновой нагрузки.

## ✨ Добавлено
- **Живое автообновление списка.** Пока панель открыта, статусы контейнеров меняются **в реальном времени**,
  новые появляются, удалённые исчезают — без нажатия ⟳. Обновление точечное, поэтому прокрутка и раскрытые
  группы сохраняются.

## 🐛 Исправлено
- **Групповые действия по делу.** «Старт / Пауза / Стоп всех» применяются только к подходящим контейнерам — без
  лишних ошибок.
- **Podman: список образов.** Образы больше не дублируются, дата создания показывается по-человечески.
- **Меньше «морганий».** Разовый сбой опроса больше не стирает список — последнее корректное содержимое остаётся.

## ⚡ Производительность
- **Лёгкое автообновление.** Частый опрос берёт только быстро меняющиеся данные (контейнеры/поды); тяжёлый
  расчёт диска и списков образов/томов — реже. Меньше фоновой нагрузки.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
