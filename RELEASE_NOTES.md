**LiteEditorAI v1.0.101** — лёгкий редактор для работы с ИИ-агентами в терминале.

Большое обновление: у редактора появилась **экосистема модулей справа от терминала** — Код, Git и новый модуль управления контейнерами.

## 🐳 Новый модуль «Контейнеры» — Docker и Podman прямо в редакторе

Полноценная замена десктопным GUI, не выходя из рабочего окна. Открывается из меню **«Модули»**.

- **Вкладки Docker | Podman** с автоопределением: показывает версии `docker` / `docker compose` / `docker-compose` (и так же для podman). Если движок не установлен — честно об этом пишет.
- **Контейнеры по compose-проектам** — сворачиваемые группы с цветным градиентом, **сортировкой стрелками** (порядок запоминается) и **групповыми действиями**: старт / пауза / стоп / удалить **всю группу** разом.
- **Поды** (Podman), **Образы**, **Тома** и полоска **занятого места на диске**.
- Управление каждым объектом из UI: старт · пауза · возобновление · перезапуск · стоп · удаление.
- **Клик по контейнеру** → детальный вид прямо в панели: **живые логи** и **встроенный терминал внутрь контейнера** (`exec` — сам выбирает bash или sh).

## ⎇ Git теперь боковая панель (в стиле JetBrains)

Вместо модального окна — удобная панель справа на выбранный проект: ветки и переключение, изменения с **диффом по клику**, поле коммита (Commit / Commit & Push / Fetch / Pull / Push), откат файла и **история коммитов**. При переключении проектов панель следует за активным.

## 🧩 Единый правый слот для модулей

Вспомогательные панели — **Код** (вивер), **Git**, **Контейнеры** — открываются по одной за раз справа от терминала и собраны в меню **«Модули»**. Терминал остаётся главным, окно аккуратно подстраивается.

## 🔧 Меню причёсано

Осталось **Файл · Настройки · Модули · О программе**. «Логи» переехали в «Файл»; «OpenRouter» и «Пульт» — в «Настройки».

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
