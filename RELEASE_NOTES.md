**LiteEditorAI v1.1.67** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Большое обновление вивера («Проект»): дифф в две колонки, локальная история файлов, замена по
проекту, git-догонка до уровня привычных IDE, автодополнение, blame-аннотации и подсветка всех языков.**

## ✨ Добавлено
- **Side-by-side дифф** — изменения файла показываются двумя колонками со сцепленным скроллом и
  внутристрочной подсветкой; тумблер «Рядом / Единый», выбор запоминается.
- **Локальная история файлов** — версии копятся при сохранениях и внешних правках (агент, git);
  дифф любой версии против текущего файла и откат. Страховка от правок агента без коммита.
- **Замена по проекту** (Ctrl+Shift+R) — поиск с чекбоксами, регистр/regex (`$1`-группы),
  «Заменить (N)»; перед записью каждого файла — снапшот в локальную историю.
- **Git**: галочка **Amend**, **история файла** (переживает переименования), cherry-pick / revert /
  копирование хеша и сообщения из контекст-меню коммита в истории.
- **Автодополнение по словам документа** — попап по мере ввода + Ctrl+Space.
- **Blame-аннотации** — колонка «автор · возраст» слева по всем строкам.
- **Подсветка всех языков** — PHP, Go, Rust, YAML, Shell, Dockerfile и сотни других, в том числе
  в диффах и модалке конфликтов.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
