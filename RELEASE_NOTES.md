**LiteEditorAI v1.0.179** — лёгкий редактор для работы с ИИ-агентами в терминале.

Новый модуль **«Инструменты»** — локальный девелоперский комбайн прямо рядом с терминалом, плюс фикс
в модуле «Контекст».

## 🔧 Новый модуль — «Инструменты»

Набор повседневных девелоперских утилит в правом слоте. **Всё считается локально, без сети** — ничего
никуда не отправляется. Сверху — фильтр по названию, ввод каждого инструмента сохраняется между сессиями.

- **Base64 ↔ картинка** — перетащите изображение (или выберите файл) и получите готовый `data:`-URI и «сырой»
  base64 с превью, размерами и весом; в обратную сторону — вставьте base64 / `data:`-URI и получите превью
  с возможностью скачать (тип определяется по сигнатуре: PNG, JPEG, GIF, WebP, BMP, ICO, SVG).
- **JSON-вьюер** — сырой JSON превращается в дерево со **сворачиванием групп**, подсветкой по типам и
  счётчиком элементов; «развернуть / свернуть всё», копирование форматированного JSON, точная позиция
  ошибки парсинга.
- **Кодеки** — Base64 (с url-safe), URL, Hex (+ hex-dump), HTML-сущности.
- **Данные** — JSON ↔ YAML / pretty / minify, Query-string ↔ JSON, CSV ↔ JSON, JSONPath.
- **Крипто** — хэши MD5 / SHA-1 / SHA-256 / SHA-512, JWT-декодер с разбором сроков действия.
- **Время** — Unix-timestamp во все форматы (+ относительное время), cron с расшифровкой и ближайшими запусками.
- **Текст** — регистр и транслит (camelCase / snake_case / kebab-case…), операции над строками, regex-тестер
  с подсветкой совпадений, diff двух текстов, генераторы Lorem и фейк-данных.
- **Цвет** — конвертер HEX ↔ RGB ↔ HSL с превью и альфа-каналом.

## 🐛 Исправлено

- **Модуль «Контекст»** теперь корректно подхватывает внешние правки `CLAUDE.md` / `AGENTS.md` — изменения,
  сделанные вне редактора, видны без ручной пересборки графа.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).
