**LiteEditorAI v1.1.37** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Модуль «Базы данных» дорастает до уровня DataGrip.** Полный пересмотр работы со схемой, данными и объектами.

## ✨ Добавлено
- **DDL объектов** — просмотр исходного определения таблиц, представлений и других объектов схемы.
- **Связи (внешние ключи) и ER-диаграмма** — наглядная карта зависимостей между таблицами.
- **Inline-редактирование ячеек** прямо в сетке данных.
- **Транзакции** — выполнение пакета операторов одной транзакцией.
- **Экспорт и полная выборка** больших таблиц помимо постраничного просмотра.
- **Отмена выполняющегося запроса**, переподключение и пинг соединения.
- **Обзор объектов схемы** с информацией об объекте и колонках.
- **Фильтрация и сортировка** данных таблицы (`WHERE`, `ORDER BY`) прямо из интерфейса.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
