**LiteEditorAI v1.1.70** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Свой порядок карточек в «Избранном» перетаскиванием, черновик коммита больше не теряется при
перезапуске, фикс сохранения настроек git-панели + README приведён в соответствие с редактором.**

## ✨ Добавлено
- **Ручной порядок «Избранного»** — карточки проектов в группе «Избранное» переставляются
  перетаскиванием, порядок сохраняется между перезапусками; снятие ★ сбрасывает позицию.
- **Черновик сообщения коммита** (окно «Проект», секция «Коммит») переживает перезапуск
  редактора — набранный текст не пропадает, пока не закоммитите.

## 🐛 Исправлено
- **Избранные ветки** (⭐ во вкладке «Ветки») и **высота панели «Ветки · История»** окна «Проект»
  сбрасывались при перезапуске редактора — теперь сохраняются.

## 📄 Документация
- **README сильно актуализирован**: «Код» и «Git» объединены в описание окна **«Проект»**
  (side-by-side дифф, локальная история файлов, замена по проекту, amend / cherry-pick / revert /
  blame, подсветка всех языков), добавлены отсутствовавшие модули — **ИИ компания, Помодоро,
  Монитор ресурсов, Сейф паролей, Мониторинг сайтов**, вкладка «Анализ диалогов» в «Контексте»,
  канбан и подзадачи в «Задачах», заставка-«матрица».

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
> Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
