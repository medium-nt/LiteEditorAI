**LiteEditorAI v1.0.157** — лёгкий редактор для работы с ИИ-агентами в терминале.

Новый модуль **«WEB/SEO аудит»** — самостоятельный анализатор сайтов. Проверяет и **локально запущенный**
dev-сервер, и **внешний домен**; ведёт свой список сайтов с **историей аудитов**, а найденные проблемы можно
одной кнопкой **передать агенту** для исправления. Встроенный Chromium рендерит страницу по-настоящему —
поэтому видны и метрики скорости, и SEO современных SPA-сайтов.

## 🔒 Безопасность

Заголовки (CSP, HSTS, X-Frame-Options, …) с оценкой и советом, **TLS-сертификат** (издатель, срок, шифр,
цепочка, доверие), куки, обнаружение **открытых `.git/` и `.env`**, утечки версий ПО, mixed content.

## 🔍 SEO

Title / description / canonical / viewport / lang, OpenGraph и JSON-LD, структура заголовков, картинки без
`alt` и размеров — **из отрендеренной страницы** (видит контент, добавленный JavaScript). Плюс проверка
**битых ссылок** и наличие robots.txt / sitemap.xml / security.txt.

## ⚡ Скорость

**Core Web Vitals** (LCP, CLS, FCP, TTFB) и тайминги, **вес страницы** и число запросов с разбивкой по типам,
тяжёлые ресурсы, ошибки в консоли, **скриншоты** десктоп и мобильный.

## 🌐 Сеть и домен

Цепочка редиректов, DNS-записи, почтовая гигиена **SPF / DMARC**, **WHOIS** (возраст домена, регистратор,
срок), **гео-IP** и хостер.

## 🤖 Связка с агентом

Оценки Безопасность / SEO / Скорость (0–100) на «Обзоре», сводка в буфер, **экспорт отчёта** `.md` / `.json` и
кнопка **«Передать агенту»** — находки уходят в терминал как задача на исправление. Результаты приходят
**поэтапно**: быстрый скан виден сразу, скриншоты, метрики и проверка ссылок досыпаются фоном.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android. **В этом релизе пульт не менялся** — подойдёт APK из
  прошлого релиза; если он уже установлен, переустанавливать не нужно.

> Alpha: терминалы не переживают перезапуск редактора. Связь пульта идёт через релей, который видит трафик (не end-to-end). Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
