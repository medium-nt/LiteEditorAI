**LiteEditorAI v1.1.78** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Новый модуль «Kafka» — топики, консюмер-группы с лагом, peek и продюс сообщений,
live-tail топика и живые графики; связка «Контейнеры» → Kafka в один клик.**

## ✨ Добавлено
- **Модуль «Kafka»** (клиент кластера поверх kafkajs): профили с категориями и меткой
  **PRODUCTION** (SASL PLAIN / SCRAM-SHA-256/512, TLS; пароли — в системном хранилище ключей),
  **вкладки** — несколько кластеров одновременно; воркспейс **Обзор / Топики / Группы / Live-tail**.
- **Обзор кластера**: брокеры с контроллером, счётчики, **живые графики** скоростей
  записи-чтения и **суммарного лага консюмер-групп** (скорости считаются из оффсетов —
  у Kafka нет истории метрик).
- **Топики**: партиции / RF / **ISR-здоровье** (тревога UR), **спарклайны In/с** в строках;
  создание (партиции, RF, retention, cleanup.policy), удаление, **очистка через DeleteRecords**,
  добавление партиций, правка retention, конфиги и таблица партиций.
- **Peek сообщений**: последние или первые N — через эфемерную группу **без следов**
  (оффсеты настоящих консюмеров не двигаются); ключ, партиция/оффсет, headers, pretty-JSON.
- **Продюс**: топик, ключ, партиция, headers, история отправок; в ответе — партиция и оффсет.
- **Группы с лагом**: состояние, участники, суммарный лаг и **тренд лага** (↑ растёт — тревога),
  бейдж «лаг без консюмеров», детали по партициям, **сброс оффсетов** в начало/конец, удаление.
- **Live-tail топика** — поток с текущего момента: фильтр по key/value, пауза с доигрыванием,
  JSON по клику; **живёт в фоне** у неактивной вкладки (индикатор ▶).
- **Связка «Контейнеры» → «Kafka»**: детект Kafka / Redpanda (образ или порты 9092/29092/9094),
  разбор advertised.listeners → адрес брокера сам подставляется — **один клик** от контейнера
  до воркспейса.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
