**LiteEditorAI v1.1.89** — лёгкий редактор для работы с ИИ-агентами в терминале.

**«Контейнеры» открывают веб-интерфейс запущенного контейнера одним кликом — локально и на
удалённом хосте через SSH-туннель; Podman стал вкладкой по умолчанию; у редактора и каждого окна
модуля появилась тонкая рамка, чтобы окна не сливались на тёмном фоне.**

## ✨ Добавлено
- **«Контейнеры» → «Открыть в браузере»**: у запущенного контейнера с опубликованными портами —
  кнопка, открывающая его веб-интерфейс. Один порт открывается сразу, несколько — выпадашкой с
  подписями портов (веб-порты вперёд); URL можно скопировать из списка или по правому клику.
  Кнопка есть и в списке, и в шапке карточки контейнера.
- **То же на удалённом хосте**: веб-порт контейнера сервера открывается через **SSH-туннель** —
  в том числе порт, слушающий только localhost сервера. На строке с активным туннелем — значок-цепочка
  (клик закрывает туннель), а клик по бейджу хоста показывает список активных туннелей с закрытием.
- **Выбор движка, когда на хосте есть оба сокета** (Docker и Podman): модалка выбора, решение
  запоминается для этого хоста. Если Podman установлен, но его API-сокет спит, — подсказка с кнопкой
  включения по SSH.
- **«Рамка окна»**: деликатная тонкая обводка по периметру редактора и каждого окна модуля, чтобы
  тёмные окна не сливались с тёмным фоном. В настройках — включение, палитра из 8 оттенков,
  необязательная пульсация с настраиваемым периодом (2–30 с) и живой предпросмотр. По умолчанию
  включена (мягкий тёмно-зелёный, плавная пульсация). Окна модулей подхватывают настройку на лету.

## 🔄 Изменено
- **Вкладка контейнеров по умолчанию — теперь Podman**, а не Docker; выбор запоминается. На
  неактивной-по-умолчанию вкладке — ссылка **«сделать вкладкой по умолчанию»** (закрепляет движок
  первым и открываемым при старте модуля).

## 🔒 Безопасность
- **Упрочнены временные (staging) файлы вивера**, куда попадают SQL-выгрузки и файлы, скачанные с
  хостов: каталог в `/tmp` создаётся с правами `0700`, файл — `0600`, имя каталога — из криптостойкого
  источника. Соседний локальный пользователь больше не может их прочитать.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
