**LiteEditorAI v1.1.41** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Переработка модуля «Обработка текста» — рабочий стол с чатом.** Раньше AI-правка фрагмента всплывала модальным окном. Теперь это полноценный рабочий стол: **документ слева, чат с агентом справа** — «текст и общение» рядом. Плюс история правок с откатом, формулы, нумерация по абзацам, закладки и панель форматирования.

## ✨ Добавлено
- **Чат с агентом вместо модалки**: выделите фрагмент → он попадает в контекст справа (можно закрепить) →
  напишите инструкцию → ответ приходит в ленту. Без выделения агент работает со всем документом. Панель
  сворачивается и тянется по ширине.
- **Замена по ответу**: у каждого ответа — «Заменить выделенное» / «Вставить ниже» / «Копировать» /
  «Ещё раз» (несколько прогонов копятся версиями).
- **История изменений документа**: каждый применённый ответ снимает снапшот — любую правку можно
  **откатить** одной кнопкой.
- **Формулы (MathJax)**: `$…$` и `$$…$$` отображаются в превью.
- **Нумерация по абзацам**, **закладки** с быстрым переходом и **панель форматирования** как в обычных
  редакторах (жирный/курсив/заголовки/списки/цитаты/ссылки/таблицы/формулы).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
