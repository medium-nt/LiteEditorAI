**LiteEditorAI v1.1.32** — лёгкий редактор для работы с ИИ-агентами в терминале.

Крупный апгрейд модуля **«Задачи»**, улучшения вивера и фикс Git-панели.

## ✨ Добавлено
- **Модуль «Задачи» — большое обновление:**
  - **Канбан-вид** (колонки *К выполнению / В работе / Готово*) рядом со списком; переключатель сам расширяет окно.
  - **Поиск** по тексту задач и подзадачам.
  - **Drag-and-drop**: в списке — порядок задач, в канбане — смена статуса перетаскиванием.
  - **Заголовок-строка**: первая строка задачи всегда видна, длинное тело (вставленный промпт) сворачивается.
  - **Сортировка по важности**, **подзадачи-чеклист** с прогрессом, **Markdown-превью** тела и **«Копировать текст»**.
  - **«Очистить выполненные»** одной кнопкой и **бейдж счётчика** активных задач на квикбаре.
- **Вивер**: кнопки превью (Markdown/HTML) переехали в нижний бар под колонкой табов (как в PhpStorm);
  для HTML — «во весь экран» и «открыть в браузере».

## 🐛 Исправлено
- **Git-панель** — в выпадашке веток пропал мусорный «%x1f» в конце имён.
- **Вивер** — Markdown/HTML открываются исходным кодом, а не сразу превью.
- **Задачи** — список больше не мерцает и не теряет позицию прокрутки при правках.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
