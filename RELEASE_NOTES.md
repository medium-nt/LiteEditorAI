**LiteEditorAI v1.1.27** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Вивер кода становится полноценной рабочей зоной проекта.** Git переехал прямо в окно вивера
(в стиле PhpStorm), а сам вивер оброс привычными по IDE возможностями: быстрым открытием файлов,
поиском по проекту, навигацией, закладками и структурой файла.

## ✨ Добавлено
- **Git встроен в окно вивера.** Отдельного окна Git больше нет — ветки, изменения и коммиты живут в трёх
  секциях окна **«Проект»**: **Файлы**, **Коммит**, **Ветки**. Дифф открывается в центре вивера, рядом с
  кодом; коммит и переключение веток сразу обновляют бейджи проектов в редакторе.
- **Быстрое открытие файла (Ctrl+P)** — поиск файла по имени по всему проекту, с историей недавно открытых
  файлов при пустом запросе (как «Go to File» в IDE).
- **Поиск по проекту** — по содержимому файлов, прямо из окна вивера.
- **Сравнение двух файлов** — выбор пары и просмотр их диффа.
- **Навигация назад/вперёд** по открытым файлам и **закладки на строки** (звёздочка в гаттере, сохраняются
  по проекту между сессиями).
- **Хлебные крошки и структура файла**, нижняя **статус-строка** (позиция курсора, число строк, язык),
  **маркеры отступов** и **сворачивание кода**.

## 🔧 Изменено
- **Единое окно «Проект»** объединяет вивер кода, дерево файлов и Git: в квикбаре, меню и палитре теперь
  один пункт **«Проект — вивер, дерево, Git»** вместо отдельных «Вивер» и «Git».
- **Окна модулей спрашивают про несохранённые изменения перед закрытием** (✕, Alt+F4, средствами ОС).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
