**LiteEditorAI v1.0.53** — лёгкий редактор для работы с ИИ-агентами в терминале.

### 🐛 Исправлено
- **Эмодзи в статус-строке Claude Code больше не «наезжают» на текст** ([#1](https://github.com/DanielLetto2020/LiteEditorAI/issues/1)).
  Терминал переключён на таблицы ширин **Unicode 11** — эмодзи (📁, ⏰ и др.) теперь корректно занимают 2 ячейки.
  Спасибо @Eurgen за детальный баг-репорт.

### 💬 Напоминание о свежем (v1.0.52)
- Чат с моделями через **OpenRouter** по своим ключам: выбор модели с ценой и контекстом, ответы стримом,
  Markdown + подсветка кода, сессии на ключ, показ и скачивание сгенерированных изображений, баланс ключа на плашке.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуй и запусти `LiteEditorAI.exe` (установка не нужна).

> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
