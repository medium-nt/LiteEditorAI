**LiteEditorAI v1.1.54** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Терминал на macOS наконец чинится по-настоящему (особенно на Intel) + новый модуль «Монитор
ресурсов».**

## ✨ Добавлено
- **Модуль «Монитор ресурсов»** (меню «Модули»). Показывает, сколько памяти и CPU потребляет сам
  редактор — **отдельно по процессам** (главное окно, каждое окно-модуля, GPU, ядро) и **отдельно**
  процессы агентов в терминалах (полезная нагрузка). Сводка, живой график памяти, кнопка «скопировать
  снимок». Помогает понять, что именно расходует ресурсы.

## 🐛 Исправлено
- **Терминал на macOS снова запускается — теперь и на Intel-Mac.** Вспомогательный бинарник терминала
  (`spawn-helper`) поставлялся **без флага исполнения**, из-за чего на Intel-Mac терминал падал с
  `posix_spawnp failed` при любом шелле. Сборка теперь проставляет флаг исполнения и подпись нужным
  бинарникам (завершает начатое в v1.1.53, где ставилась только подпись).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
