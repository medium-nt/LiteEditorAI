**LiteEditorAI v1.1.42** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Большая доработка модуля «Базы данных»: грид уровня DataGrip + новый инструмент «AI-DB» — разговор с базой через ИИ-агента.** Сетка данных доведена до уровня настольных SQL-клиентов, а рядом появился чат, который понимает обычный язык, сам пишет запросы и красиво показывает результат — **строго на чтение, без единой записи в базу**.

## ✨ Главное — AI-DB (чат с базой)
Внутри подключения — вторая вкладка **«AI-DB»**: слева дерево таблиц, справа чат. Спрашиваете на обычном языке — агент, зная структуру всех таблиц, формирует `SELECT`, **показывает его и ждёт вашего подтверждения**, после «Выполнить» приложение исполняет запрос и выводит результат **текстом, таблицей или графиком**.

- **Любая модель:** локальные **Claude** / **Codex** (CLI), а также по API — **OpenRouter** (подтягивает список моделей с контекстом и ценами), **Ollama** и **LM Studio** (локальные модели — данные не уходят с машины). Настройка моделей глобальная, в отдельной модалке.
- **Стриминг** ответов токен-за-токеном + индикатор печати.
- **Сессии чата:** боковая колонка, закрепление, группировка по датам, удаление с подтверждением, сохранение между запусками.
- **Уточнения** в стиле терминала Claude: список плашек с кастомными чекбоксами/радио, поле «Свой ответ» и кнопка «Отправить» (одиночный/множественный выбор).
- **Графики (Chart.js):** 8 типов, под тему приложения, **экспорт в PNG** — агент отдаёт готовую картинку.

## 🔒 Безопасность — строго только чтение
- AI-DB **никогда не пишет в базу**, и это держится **не на доверии к агенту**: каждый предложенный запрос проходит фильтр изменяющих операторов (`INSERT/UPDATE/DELETE/DROP/ALTER/CREATE/TRUNCATE/COPY/CALL/ATTACH`, в т.ч. `SELECT … INTO …`) **плюс** белый список (запрос обязан начинаться с `SELECT/WITH/EXPLAIN/SHOW`). Кнопка «Выполнить» блокируется для всего остального. Агент **не обращается к базе напрямую** — единственный исполнитель запросов само приложение.
- Ответы агента **санируются** (защита от XSS), даже когда внутри эхо данных из базы.
- Ключи и адреса провайдеров хранятся **локально**; локальные модели держат всё на вашей машине.

## ✨ Также добавлено в грид
- **Таймер автообновления** таблицы (пауза/5с/10с/30с/1мин/5мин/свой), **локальный кеш вкладок** (переключение не делает лишних запросов).
- **Просмотр значения** — выезжающая панель с деревом JSON (номера строк, сворачивание, метрики), живое обновление по выбору ячейки.
- **Менеджер столбцов** (скрыть/показать + перетаскивание), **горизонтальный скролл** всегда внизу, надёжный **ресайз колонок**.
- **Экспорт** в 8 форматов (CSV/TSV/JSON/JSONL/SQL/Markdown/HTML/XML) с выбором папки и «Копировать в буфер».
- **Категория подключения** — выбор из существующих или создание новой; модалка подключения шире и аккуратнее.

## 💡 Примеры
- «Покажи 10 самых прокатываемых фильмов» → агент пишет `SELECT … LIMIT 10` → «Выполнить» → таблица.
- «Построй график выручки по месяцам за 2023» → `SELECT` + график → картинка с «Скачать PNG».
- «Какие поля у таблицы customer?» → ответ текстом по схеме.
- Подключить локальную модель: **⚙ Настроить модели… → Ollama → Проверить и загрузить → отметить** → в выпадашке агента появится «Ollama …».

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).
