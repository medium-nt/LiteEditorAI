**LiteEditorAI v1.0.152** — лёгкий редактор для работы с ИИ-агентами в терминале.

Этот релиз — полная переработка модуля **«Контекст»** (сборка `CLAUDE.md` / `AGENTS.md` из блоков на канве):
распил стал точным и без потерь, собранный файл — чистым, появились группы из заголовков, восстановление
версий и отслеживание правок файла вне модуля.

## ✂ Распил по заголовкам — точно и без потерь

Любой markdown-заголовок (`#`…`######`) становится отдельным блоком, а обратная сборка даёт исходный файл
**без искажений** (проверено побайтно). Распил выполняется мгновенно, без обращения к ИИ. Распилить можно как
весь файл, так и отдельный блок (синий значок ✂ на блоке, если в нём несколько заголовков).

## 🧹 Чистый файл контекста

Собранные `CLAUDE.md` / `AGENTS.md` больше **не содержат служебных маркеров** — это обычный markdown.
Принадлежность файла модулю определяется по контрольной сумме, а не по пометкам в тексте.

## 🗂 Группы из заголовков

Заголовок-контейнер (без своего текста, но с вложенными под-заголовками) при распиле автоматически становится
**группой** — вложенность любой глубины. Два вида: «обёртка» (только переключатель) и «группа-текст» (пишет свой
заголовок в файл). Несколько блоков можно сгруппировать вручную — выделите и нажмите `Ctrl+G`.

## 🕘 Восстановление версий

Любую версию из «🕘 Версии» можно вернуть на канву одним блоком или назначить **новым «Оригиналом»** (базовый
снимок обновляется, прежний остаётся обычной версией).

## 👀 Отслеживание правок файла вне модуля

Если файл изменён снаружи (например, агентом) — в шапке загорается значок, а окно сверки покажет новые,
изменённые и удалённые разделы. Для конфликтов — выбор **«Оставить моё / Взять из файла / Соединить»**.
Сборка больше **не затирает внешние правки молча**.

## 🔧 Доводка интерфейса

- **Профиль** одной строкой: имя + значок активного профиля, действия в меню «⋯» (переименовать, дублировать,
  **сохранить полный файл**, удалить).
- Заголовок текстового блока берётся из его содержимого автоматически.
- «Подтвердить» / «Сбросить» — компактные иконки; выделение блоков рамкой (`Shift`+протягивание) и групповое
  перемещение.

## ⚠️ Важно при обновлении

- При первой сборке после обновления файл может **однократно** отметиться как «изменён вне модуля» (из-за перехода
  на чистый формат без маркеров) — нажмите «Перезаписать», дальше всё синхронно.
- Набор блоков по-прежнему — **«Текст»** и **«Группа»** (файл / команда / память агента вернутся позже).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android. **В этом релизе пульт не менялся** — подойдёт APK из
  прошлого релиза; если он уже установлен, переустанавливать не нужно.

> Alpha: терминалы не переживают перезапуск редактора. Связь пульта идёт через релей, который видит трафик (не end-to-end). Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
