**LiteEditorAI v1.1.14** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Каждый модуль — в отдельном окне.** Крупное изменение интерфейса: панели-модули (Git, вивер кода,
контейнеры, базы данных, задачи и другие) больше не теснятся в правом слоте редактора — теперь каждый из них
открывается как самостоятельное окно, которое можно свободно расположить на экране.

## ✨ Добавлено
- **Модули как отдельные окна.** Любой встроенный модуль открывается в своём окне: можно держать открытыми
  сразу несколько (например, Git и базы данных рядом), повторный клик переводит фокус на уже открытое окно.
  Каждое окно **запоминает свой размер и положение**, а набор открытых окон **восстанавливается при следующем
  запуске** редактора.
- **Окна следуют за проектом.** Git, Контекст, Задачи, Аудит и вивер кода с деревом файлов автоматически
  показывают данные активного проекта — переключаете проект, окна обновляются сами. Самостоятельные модули
  (Базы данных, Контейнеры, Удалённые хосты, WEB/SEO аудит, Инструменты, IterFlow, OpenRouter, Обработка
  текста) от проекта не зависят.
- **Вивер кода и дерево файлов — отдельное окно**, следующее за активным проектом. Открыть файл можно как
  раньше — кликом по пути в терминале или из модуля; дифф, превью Markdown/HTML/картинок, миникарта и
  сохранение по Ctrl+S работают как прежде.
- **Системный терминал** «Система · ~» тоже стал отдельным окном.

## 🔧 Изменено
- **Закрытие редактора закрывает все окна модулей** — корректно, с освобождением памяти.
- Правый слот редактора теперь занимают только **«Мои модули»** (пользовательские плагины).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
