**LiteEditorAI v1.0.103** — лёгкий редактор для работы с ИИ-агентами в терминале.

Небольшое, но приятное удобство ввода в терминале — по просьбе из Issues.

## ⌨️ `Ctrl+Enter` — перенос строки в терминале

Раньше `Enter` в терминале всегда сразу выполнял команду, а перенести ввод на новую строку можно было только через `Alt+Enter`. Теперь это умеет и **`Ctrl+Enter`** — он продолжает ввод на следующей строке, не запуская команду:

- на **Linux / macOS** — через `\` + перевод строки (вторичное приглашение `>` в bash/zsh);
- на **Windows** — через `LF` (ConPTY / PSReadLine, приглашение `>>`).

Работает и в проектных, и в системных терминалах.

| Клавиша | Действие |
|---|---|
| `Enter` | выполнить команду |
| `Ctrl+Enter` | перенос строки, продолжение ввода |

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
