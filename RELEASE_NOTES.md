**LiteEditorAI v1.0.100** — лёгкий редактор для работы с ИИ-агентами в терминале.

### 🐛 Исправлено
- **Окно не возвращало прежний размер после закрытия вивера** (#3). Если окно было растянуто почти на весь
  экран, открытие вивера упиралось ростом в край экрана, а закрытие всё равно ужимало окно на полную ширину
  панелей — и оно становилось **уже исходного**. Теперь рост и сжатие симметричны: окно возвращается ровно
  к размеру, что был до открытия вивера (ручной ресайз между этими действиями учитывается). То же — для
  системного («скретч») терминала. Спасибо @Eurgen за репорт.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуй и запусти `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установи на устройство (разреши установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
