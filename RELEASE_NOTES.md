**LiteEditorAI v1.0.194** — лёгкий редактор для работы с ИИ-агентами в терминале.

Вылизывание встроенного модуля **«Git»** к релизу: убран мёртвый код, исправлены логические шероховатости,
просмотр диффа стал полнее.

## ✨ Добавлено
- **Дифф нового файла.** Клик по новому (ещё не отслеживаемому) файлу показывает его содержимое как
  добавленные строки, а не пустую панель.

## 🐛 Исправлено
- **Коммит ровно выбранного.** Выборочный коммит берёт строго отмеченные файлы — посторонние изменения из
  индекса больше не «прилипают» к коммиту.
- **Галка не прилипает.** Файл, ранее снятый с коммита, при повторном изменении снова включается автоматически.
- **Надёжный разбор конфликтов.** Окно разрешения слияния корректно обрабатывает маркеры, включая diff3-стиль.
- **Имя новой ветки.** Недопустимые имена отклоняются с понятной ошибкой.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
