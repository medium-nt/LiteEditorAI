**LiteEditorAI v1.0.195** — лёгкий редактор для работы с ИИ-агентами в терминале.

Доработка встроенного модуля **«Базы данных»** (PostgreSQL / MySQL / SQLite): устойчивость к обрывам
соединения, надёжное сохранение SQLite и ряд исправлений в SQL-консоли, экспорте и форме подключения.

## 🐛 Исправлено
- **Устойчивость к обрыву соединения.** При неожиданном разрыве связи с базой (тайм-аут простоя, перезапуск
  сервера, сетевой сбой) редактор больше не закрывается — соединение переустанавливается автоматически при
  следующем запросе.
- **Надёжное сохранение SQLite.** Изменения, выполненные вместе с выборкой в одном запуске
  (`INSERT … ; SELECT …`), теперь гарантированно записываются в файл базы.
- **Режим «только чтение».** Запросы со строковой функцией `REPLACE()` больше не отклоняются как изменяющие.
- **Экспорт.** При ошибке загрузки таблицы экспорт больше не сохраняет по ошибке данные предыдущей таблицы.
- **Форма подключения.** При смене типа базы порт по умолчанию подставляется корректно; введённый вручную порт сохраняется.
- **Постраничный просмотр.** Кнопка «Вперёд» больше не ведёт на пустую страницу.
- **MySQL.** Результат скрипта из нескольких выражений отображается корректно.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
