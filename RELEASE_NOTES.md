**LiteEditorAI v1.1.53** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Терминал на macOS снова работает + модуль «Контекст» научился извлекать правила из истории Ваших
диалогов с агентом.**

## ✨ Добавлено
- **Модуль «Контекст» → вкладка «Анализ диалогов».** Просматривает историю Ваших диалогов с агентом
  (Claude Code) по текущему проекту и извлекает из неё **долгоиграющие правила** — то, что стоит закрепить,
  чтобы агент не забывал. Каждое правило сгруппировано по рекомендации, **куда его положить** (главный
  `CLAUDE.md`, проектный `CLAUDE.md`, `AGENTS.md`, память или «на ревью»). Есть поиск, копирование и экспорт
  в JSON. ⚠️ Инструмент **ничего не пишет** в реальные файлы — только собирает и показывает; решение за Вами.

## 🐛 Исправлено
- **Терминал на macOS снова запускается.** На сборках без подписи Apple вспомогательный бинарник терминала
  не запускался — терминал падал с ошибкой `posix_spawnp failed` (особенно на Apple Silicon). Теперь при
  сборке бандлу проставляется ad-hoc-подпись, и терминал работает.

## 🔄 Изменено
- **Снимки сессий между перезапусками убраны.** Восстановление прошлого вывода терминала над свежей сессией
  на практике больше мешало, чем помогало. Между перезапусками сохраняются только **имена вкладок**; ранее
  накопленные снимки автоматически вычищаются из хранилища.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
