## ✨ Добавлено
- **Obsidian-сайдбар**: дерево документов активного проекта (`.md`/`.txt`/`.docx`) — ленивая
  подгрузка папок, живой поиск, сортировка A↕Я, создание файлов и папок; дерево следит за
  диском и обновляется, когда агент правит файлы.
- **Вкладки документов** с маркером несохранённых изменений; **нативные диалоги**
  «Открыть» / «Сохранить как».
- **Двухрежимный редактор**: «Разметка» (WYSIWYG-страница «как в Word» — белая бумага
  независимо от темы) ⇄ «Markdown» (исходник); плавающий пилл форматирования: стили абзаца,
  **B/I/U**, выравнивание, списки, две колонки, таблицы, зум и интервал, пинч-зум тачпадом.
- **Формулы LaTeX на KaTeX**: палитра символов, живое превью, инлайн/блок с нумерацией —
  локально, без CDN.
- **Живой стриминг ответа агента** в чат — текст появляется по мере генерации.
- **Роли агента per-project** (`Roles/*.md`): выбор чипом, автозаготовки
  Редактор/Корректор/Переводчик/Юрист.
- **Gemini** — третий агент чата (официальный `gemini` CLI) рядом с Claude и Codex.
- **Автосейв** файла через 1,5 с после правки; закрытие несохранённой вкладки/окна —
  с подтверждением; `Ctrl+S` — ручное сохранение.
- **README: раздел «Благодарности»** — вклад сообщества по категориям.

## 🔄 Изменено
- Модуль «Обработка текста» полностью переработан; титлбар модульных окон — в стиле macOS.
- Прежние документы модуля (`~/.LiteEditorAI/textproc/`) остаются на диске — открывайте их
  через «Открыть» или дерево проекта (списка старой библиотеки в новом интерфейсе нет).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).
