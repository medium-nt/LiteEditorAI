**LiteEditorAI v1.1.55** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Вкладка «Анализ диалогов» (модуль «Контекст») доведена до рабочего инструмента: разбор длинной истории
частями, накопление между запусками и запись правил прямо в файлы контекста.**

## ✨ Добавлено
- **Разбор длинной истории частями.** Анализ идёт сессиями (свежие первыми) до лимита, кнопка
  **«Продолжить — ещё N сессий»** догружает следующую часть; правила накапливаются, дубли объединяются.
- **Реестр сохраняется между перезапусками**, а при появлении новых диалогов **«Продолжить» разбирает
  только новые** сессии (ранее разобранные не перечитываются).
- **Запись правил в файлы.** Отметьте правила галочками → **«Применить выбранные»** допишет их в нужный
  `CLAUDE.md` / `AGENTS.md` (с подтверждением и списком файлов). Память и «на ревью» в файлы не пишутся.
- **Дедуп с уже записанным:** правила, уже присутствующие в `CLAUDE.md`/`AGENTS.md`, помечаются «уже есть»
  (с переключателем «скрыть уже есть»).
- **Триаж карточек:** изменить (заголовок/детали/куда положить), игнорировать, копировать.

## 🎨 Интерфейс
- Карточки правил — теперь по **2 в ряд** (на узком окне — в одну колонку).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
