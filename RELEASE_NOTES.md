**LiteEditorAI v1.1.77** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Новый модуль «RabbitMQ» — очереди, live-tail и живые графики; вкладки подключений
в «Базах данных» и связка «Контейнеры» → Базы данных / RabbitMQ в один клик.**

## ✨ Добавлено
- **Модуль «RabbitMQ»** (клиент management API брокера): профили серверов с категориями,
  цветами и меткой **PRODUCTION** (пароли — в системном хранилище ключей), воркспейс
  **Обзор / Очереди / Exchanges / Подключения** с живым обновлением без потери фокуса;
  **peek сообщений без съедания**, публикация с историей и проверкой маршрутизации;
  purge / delete / закрытие подключений — с подтверждением и гардом на PRODUCTION-профилях.
- **Live-tail: прослушивание exchange в реальном времени** (такого нет в стандартном web-UI):
  временная auto-delete-очередь подписывается на exchange, живой поток сообщений с паузой
  и доигрыванием, разворот JSON по клику; кнопка ▶ прямо на строке exchange.
- **Графики и метрики в «Обзоре» RabbitMQ**: «Сообщения в очередях» (Ready / Unacked)
  и «Скорости» (publish / deliver / ack / redeliver) с диапазонами 10 мин / 1 ч / 8 ч / 24 ч;
  **спарклайны глубины** прямо в строках очередей с трендом ↑ / ↓ и бейдж **«нет консюмеров»**,
  когда сообщения копятся впустую.
- **Вкладки подключений в «Базах данных»**: несколько открытых воркспейсов одновременно,
  «+» — открыть существующее подключение или создать новое, «назад» — список без закрытия
  вкладки; набор вкладок переживает перезапуск редактора.
- **Вкладки профилей в «RabbitMQ»** — несколько брокеров сразу; live-tail у неактивной
  вкладки **продолжает работать в фоне** (кадры доигрываются при возврате, индикатор ▶).
- **Связка «Контейнеры» → «Базы данных» / «RabbitMQ»**: редактор распознаёт СУБД и RabbitMQ
  в контейнере, подключение заполняется само из его настроек — **один клик** от контейнера
  до готового воркспейса; повторный клик переключает на уже открытую вкладку.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple, поэтому при
  **первом** запуске разрешите её **без терминала**: запустите приложение → закройте предупреждение → откройте
  **Системные настройки → Конфиденциальность и безопасность** → пролистайте вниз и нажмите **«Открыть всё
  равно»**. Кому удобнее терминал — то же одной командой: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: живой процесс терминала не переживает перезапуск редактора (сохраняются только имена вкладок).
