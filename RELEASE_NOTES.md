**LiteEditorAI v1.0.104** — лёгкий редактор для работы с ИИ-агентами в терминале.

Шлифуем интерфейс: карточки проектов стали компактнее, а модуль Git — аккуратнее и функциональнее.

## 🗂 Карточки проектов компактнее

Три кнопки действий (**вивер**, **заметки**, **Git**) переехали из нижнего футера прямо в строку заголовка — сразу после звезды «избранное». Разделительная полоска и сам футер убраны, поэтому каждая карточка стала ниже, а список проектов — плотнее.

## ⎇ Модуль «Git» — аккуратнее и мощнее

**Кнопки больше не массивные.** Громоздкие кнопки заменены на компактные pill-кнопки с иконками, разложенные в две ровные строки:

- **Commit** · **Commit & Push**
- **Fetch** · **Pull** · **Push** │ **Stash** · **Pop**

**Новые операции:**

- 📦 **Stash и возврат** — спрятать все изменения (включая неотслеживаемые файлы) одной кнопкой и вернуть их обратно.
- 🧹 **«Откатить все правки»** — кнопка в заголовке секции «Изменения» (с подтверждением): отменяет изменения во всех отслеживаемых файлах, новые файлы не трогает.
- 📋 **Клик по коммиту в истории копирует его хеш** — удобно вставить в `git revert`, `cherry-pick` и т.п.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
