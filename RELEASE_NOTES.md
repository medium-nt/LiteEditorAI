**LiteEditorAI v1.0.99** — лёгкий редактор для работы с ИИ-агентами в терминале.

### 🛰 Удалённый пульт (Android)
- **Своя иконка приложения.** APK теперь ставится с фирменной иконкой (тёмный градиент + синий шеврон `>`
  и зелёный курсор `_`) вместо системной заглушки — адаптивная для Android 8+ (с монохромным слоем под
  тематические иконки Android 13+) и обычные PNG для более старых версий. Версия APK — **0.44.0**.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуй и запусти `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установи на устройство (разреши установку из неизвестных источников).

> Десктоп-часть (`.deb`/`.zip`) функционально не менялась с v1.0.98 — основное в этом релизе у пульта.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
