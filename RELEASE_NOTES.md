**LiteEditorAI v1.0.102** — лёгкий редактор для работы с ИИ-агентами в терминале.

Экосистема модулей пополнилась: теперь рядом с терминалом — ещё и **базы данных**, а системный терминал получил **вкладки**.

## 🗄 Новый модуль «Базы данных» — клиент Postgres / MySQL · MariaDB / SQLite

Лёгкая замена DataGrip для повседневных задач, прямо в окне редактора. Драйверы встроены — **ставить ничего не нужно**. Открывается из меню **«Модули»**.

- **Подключения** к PostgreSQL, MySQL/MariaDB и SQLite — **по хосту или через SSH-туннель**. Сгруппированы по категориям (сворачиваемые аккордеоны с градиентом, по умолчанию «Все»). Создание, правка, **тест соединения**, режим **«только чтение»**.
- **Дерево схемы** (схемы → таблицы и представления) и **просмотр данных** таблицы в гриде с пагинацией.
- **SQL-консоль** с подсветкой синтаксиса и выполнением по `Ctrl+Enter` — результат сразу в гриде.
- **Экспорт** результата или таблицы в **CSV / JSON / SQL** и **импорт** SQL-файла.
- **Безопасность всерьёз:** пароли (БД и SSH) шифруются системным хранилищем ключей и **не покидают бэкенд**; режим «только чтение» запрещает изменяющие запросы **на стороне сервера**; TLS по умолчанию с проверкой сертификата (доверие самоподписанному — отдельным явным флагом).

> Проверено вживую на реальных PostgreSQL 16 и MariaDB 11: подключение, чтение схемы и данных, запросы и режим «только чтение» работают.

## 🖳 Системный терминал → модуль с вкладками

Системные шеллы (вне проектов, домашняя папка) теперь живут как **модуль с несколькими вкладками**: `＋` новая, закрытие, переименование; хоткеи `Ctrl+Shift+T` / `Ctrl+Shift+W` и `Ctrl+PageUp/PageDown`. Удобно держать рядом несколько системных терминалов.

## ✨ Мелочи, которые приятно

- **Категория OpenRouter** слева теперь видна **всегда** (даже без ключей — со счётчиком и кнопкой настройки) и красиво отделена полоской «интеграции» от категорий проектов.
- **Заметки** освежили: заметная кнопка «В терминал», аккуратные иконки действий, метка очереди — функциональность та же, выглядит чище.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
