**LiteEditorAI v1.0.155** — лёгкий редактор для работы с ИИ-агентами в терминале.

Новый модуль **«Аудит»** — быстрый рентген любого проекта. Полезен и человеку (понять незнакомый проект,
найти раздувание и мусор), и агенту (паспорт проекта одной кнопкой в буфер). Источник файлов — **git-tracked**
(честный фильтр без `node_modules` и сборки) или весь каталог. Семь вкладок:

## 📊 Обзор

Паспорт проекта: файлы · строки · вес, языки по строкам, категории. **Чипы-находки** со счётчиками ведут на
нужную вкладку. Здесь же — **«Скопировать сводку»** (markdown-паспорт в буфер) и **«Экспорт отчёта»**
(полный отчёт в `.md` / `.json`).

## 🗂 Типы и 📦 Крупные

**Типы** — сортируемая таблица по расширениям (файлов · строк · вес · доля), клик по типу раскрывает его
файлы. **Крупные** — топ файлов по строкам или весу с флагом **«аномалия»** (файл сильно выше нормы для своего
типа) и фильтром «только аномалии». Отдельная вкладка **Медиа** — картинки, видео, архивы, шрифты по весу.

## 🧹 Гигиена

Мусор в гите (каталоги сборки, `.env`, временные и служебные файлы), минифицированные и длинные строки,
**дубликаты по содержимому**, возможно осиротевшие файлы (эвристика — с пометкой «проверяйте перед удалением»).

## 🚩 Долг

Метки **TODO / FIXME / HACK** и **потенциальные секреты** (ключи, токены, приватные ключи). Клик по находке
открывает файл на нужной строке.

## 🕘 История

**Горячие файлы** по git-churn (где больше всего правок — там кластеризуются баги), свежие изменения и давно не
тронутые файлы. На не-git проектах — по времени изменения файла.

## 🖱 Действия по файлу

В любом списке: открыть в вивере (клик по строке), копировать путь, отправить в терминал запрос на разбор файла.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android. **В этом релизе пульт не менялся** — подойдёт APK из
  прошлого релиза; если он уже установлен, переустанавливать не нужно.

> Alpha: терминалы не переживают перезапуск редактора. Связь пульта идёт через релей, который видит трафик (не end-to-end). Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
