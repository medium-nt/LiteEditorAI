**LiteEditorAI v1.0.198** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Пульт (Android):** управление задачами проекта прямо с телефона или планшета.

## ✨ Добавлено
- **Задачи на пульте.** В пульте появилась кнопка **«Задачи»** (рядом с «Обновить») — открывает список
  задач, зеркало панели «Задачи» из редактора: вкладки **«Проект»** и **«Общие»**, фильтры
  Активные / Все / Готово, переключение статуса и важности по тапу, изменение порядка, добавление, правка и
  удаление задач, перенос между проектом и общим списком, а также отправка текста задачи в терминал проекта
  («В терминал» и «В терминал и удалить»). Это те же задачи, что и в редакторе: правки с пульта сразу видны
  на ПК, и наоборот.

  > Нужен обновлённый пульт (**APK v0.60.0** из этого релиза) и перезапуск редактора на ПК.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (обновлён: задачи проекта).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
