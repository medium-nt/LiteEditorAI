**LiteEditorAI v1.0.199** — лёгкий редактор для работы с ИИ-агентами в терминале.

Вивер и дерево файлов: **drag-and-drop** и быстрые действия с файлами.

## ✨ Добавлено
- **Перетаскивание в дереве файлов.** Файлы и папки можно перемещать мышью между папками прямо в дереве.
  А ещё файлы и папки можно **затаскивать извне** — из системного файлового менеджера прямо в дерево: они
  копируются в выбранную папку или в корень проекта. Папка-приёмник подсвечивается; есть защита от переноса
  папки внутрь самой себя, а при совпадении имени копия получает суффикс « (2)».
- **Действия с файлом в контекстном меню дерева.** По правому клику на файле или папке добавлены
  **«Копировать файл»** (кладёт сам файл в системный буфер обмена — затем его можно вставить копией в файловом
  менеджере) и **«Показать в проводнике»** (открывает системный файловый менеджер с выделением файла).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
