**LiteEditorAI v1.1.38** — лёгкий редактор для работы с ИИ-агентами в терминале.

**Заготовленные промпты прямо в терминале.** Частые реплики агенту — в один клик из контекстного меню, без копипаста.

## ✨ Добавлено
- **Промпты в контекстном меню терминала** — правый клик → **«Промпты»**: справа выезжает список карточек,
  клик вставляет текст в активный терминал **без запуска** (можно дополнить и отправить вручную).
- **Проектные и общие** (как в «Задачах») — промпты конкретного проекта и общие для всех проектов:
  во всплывающем списке разделены группами, в менеджере — две колонки со своими скроллами и переносом между ними.
- **Менеджер промптов** (шестерёнка в шапке списка) — добавление, правка названия и текста, порядок, удаление;
  всё сохраняется автоматически.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** (win) — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🍎 **`.dmg`** — macOS: `-arm64` для Apple Silicon, `-x64` для Intel. Сборка без подписи Apple — при первом
  запуске снимите карантин: `xattr -dr com.apple.quarantine /Applications/LiteEditorAI.app`.
- 🤖 **`liteeditor-pult-0.60.0-debug.apk`** — пульт для Android (без изменений в этом релизе).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
