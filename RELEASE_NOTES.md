**LiteEditorAI v1.0.187** — лёгкий редактор для работы с ИИ-агентами в терминале.

Модуль просмотра кода стал надёжнее: превью и дифф обновляются вживую, исчезли «зависшие» панели, а у
HTML-файлов появилось «Открыть в браузере» — как в больших IDE.

## ✨ Добавлено
- **«Открыть в браузере» для HTML.** ПКМ по `.html`/`.htm` в дереве файлов → файл открывается в системном
  браузере по умолчанию. Правьте вёрстку в редакторе, сохраняйте и обновляйте вкладку в браузере.

## 🐛 Исправлено
- **Превью и git-дифф обновляются вживую**, когда файл меняется на диске (например, его правит агент).
- **Нет «зависших» оверлеев** диффа/превью при смене проекта и входе в чат; полноэкранное превью больше
  не оставляет висящую кнопку закрытия.
- **Открытие файла по ссылке из терминала** при закрытой панели просмотра больше не даёт пустое окно.
- **Дерево файлов:** папки-симлинки отображаются как папки; переименование папки обновляет путь открытого
  файла; пути со спецсимволами не ломают превью.

## 🔒 Безопасность
- Усилена очистка содержимого при рендере Markdown-превью (дополнительный рубеж поверх CSP).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
