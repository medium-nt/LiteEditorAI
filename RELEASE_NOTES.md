**LiteEditorAI v1.0.181** — лёгкий редактор для работы с ИИ-агентами в терминале.

Сервисный релиз: обновление сборочных зависимостей с уязвимостями.

## 🔒 Безопасность

- Обновлены транзитивные **сборочные** зависимости: `form-data` → **4.0.6** (CRLF-инъекция, high) и
  `tar` → **7.5.16** (file smuggling, moderate). Это dev-инструментарий (electron-builder / node-gyp), в
  поставляемое приложение он не входит — на установленную сборку не влияет; обновление закрывает алерты
  `npm audit` / Dependabot.

Функционально идентичен **v1.0.180** (фикс запуска из дистрибутива) и **v1.0.179** (модуль «Инструменты»).

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
