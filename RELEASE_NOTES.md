**LiteEditorAI v1.0.143** — лёгкий редактор для работы с ИИ-агентами в терминале.

Два больших новшества: модуль **«Контекст»** — визуальный конструктор контекста агента, и **цветной
пульт** — экран терминала на Android теперь с полной палитрой, как на ПК.

## 🧠 «Контекст» — соберите CLAUDE.md как граф

CLAUDE.md и AGENTS.md обычно растут как попало и быстро протухают. Новый встроенный модуль
**«Модули → Контекст»** превращает контекст агента в наглядный граф на канве (в духе n8n):

- 🧩 **Блоки-источники** — текст, файл, команда, слот памяти агента — соединяются проводами с выходами
  **Claude (CLAUDE.md)** и **Codex (AGENTS.md)**. На блоках и выходах — счётчики ≈токенов: видно, сколько
  «весит» ваш контекст.
- 🎛 **Профили.** Блок, подключённый через группу, — переключаемая часть контекста: тумблер выключает её
  целиком. А у проекта может быть **несколько графов-профилей** (например, «разработка» и «ревью») —
  вкладки над канвой, с индикатором несобранных изменений.
- ✂️ **Распил готового файла.** Существующий CLAUDE.md разбирается на блоки **вашим локальным агентом**
  (Claude Code / Codex — по вашей подписке, без API-ключей) и появляется на канве уже соединённым.
- 🛟 **Безопасно.** Перед перезаписью — бекап (папка и глубина настраиваются), чужой файл без вашего
  подтверждения не трогается; сборка — только по явной кнопке «Подтвердить».

## 🌈 Пульт: цветной экран терминала

Экран на Android-пульте теперь раскрашен как настоящий терминал — полная палитра (16 / 256 / RGB), фон,
жирный/курсив/подчёркивание/инверсия. Диффы агента зелёные и красные, а не серые. Трафик почти не вырос:
цвета едут внутри тех же экономных кадров-диффов, скорость пульта осталась прежней. Заодно курсор больше
не сползает на строках с эмодзи. **Нужен APK 0.54.0 из этого релиза** — старый APK продолжит работать,
но покажет экран без цветов.

## ⚡ Квикбар — модули в один клик

**«Модули → Настройка панели»**: выбранные модули (включая ваши собственные) становятся кнопками-иконками
в полосе под терминалом. Состав и порядок настраиваются и сохраняются.

## 🐛 Исправлено

- **Ctrl+F-поиск в терминале** не открывался на вкладках-сессиях.
- **Вложенные модалки** не ломали друг друга при закрытии.
- Утечка обработчиков в списке «Пульты» при закрытии окна по Esc/клику в фон.
- «Контекст»: распил падал на CLAUDE.md с код-блоками — теперь разбирается корректно, а при реальной
  ошибке агента показывается его ответ с кнопками «копировать/повторить».

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из
  неизвестных источников). **В этом релизе пульт обновился** (цветной экран) — рекомендуем переустановить APK.

> Alpha: терминалы не переживают перезапуск редактора. Связь пульта идёт через релей, который видит трафик (не end-to-end). Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
