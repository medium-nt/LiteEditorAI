**LiteEditorAI v1.0.188** — лёгкий редактор для работы с ИИ-агентами в терминале.

Сервисный релиз безопасности: закрыты все известные уязвимости в зависимостях.

## 🔒 Безопасность
- **Закрыты 9 уязвимостей** в транзитивной зависимости `undici` (HTTP-клиент сборочных инструментов
  Electron). Версии подняты до пропатченных — `undici@7.28.0` и `undici@6.27.0`. Затронуты только
  инструменты сборки; в дистрибутив, который вы скачиваете, эта зависимость не входит. `npm audit` теперь
  показывает **0 уязвимостей**.

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./LiteEditorAI_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android (без изменений с прошлого релиза).

> Alpha: терминалы не переживают перезапуск редактора. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
