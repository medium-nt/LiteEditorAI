**LiteEditorAI v1.0.109** — лёгкий редактор для работы с ИИ-агентами в терминале.

Новая интеграция **«Обработка текста»**: правьте фрагменты текста через **локального агента** — по вашей подписке Claude Code / Codex, **без API-ключей**.

## ✍️ Обработка текста

Рядом с проектами (ниже «OpenRouter») появилась категория **«Обработка текста»**. Создавалась для работы с научными статьями — изучать и править их с помощью ИИ, — но пригодится для любого текста.

- **Документы-плашки.** Каждая плашка — документ (файлы в `~/.LiteEditorAI/textproc/`): **переименовываемый заголовок** и под ним **реальное имя файла**. Создание кнопкой **＋**.
- **Текст вместо терминала.** Клик открывает документ в центре окна. Для Markdown — переключатели **Сырой / Оформленный / Совместно** (исходник, превью или две половины рядом). Изменения сохраняются автоматически.
- **Главное — кнопка «AI» над выделением.** Выделите любой фрагмент — над ним всплывёт **«AI»**. В окне можно собрать запрос из **системного промпта** (роль ИИ), **заготовленных** частых промптов и **своего** текста (достаточно одного), выбрать **агента** — и отправить фрагмент на обработку.
- **Версии результата.** Ответ приходит куском текста на замену. Прогоны **копятся версиями** (◀ ▶): **«Заменить»** применяет выбранный к выделению, **«Попробовать ещё»** генерирует новый, не теряя прошлых.
- **Без API-ключей — по подписке.** Обработка идёт **локальным агентом** (`claude -p` или `codex exec`), а не платным API. Системные и заготовленные промпты пополняются прямо в редакторе.

> Удобные сценарии для статей: «перескажи кратко», «упрости формулу», «исправь стиль».

### ⬇️ Что скачать
- 🐧 **`.deb`** — Ubuntu / Debian (x64): `sudo apt install ./liteeditor-ai_*.deb`, затем иконка **LiteEditorAI** в меню.
- 🪟 **`.zip`** — Windows (x64), portable: распакуйте и запустите `LiteEditorAI.exe` (установка не нужна).
- 🤖 **`liteeditor-pult-*.apk`** — пульт для Android: установите на устройство (разрешите установку из неизвестных источников).

> Пульт (Android) без изменений с прошлого релиза — APK можно не переустанавливать.
> Alpha: терминалы не переживают перезапуск. Баги и идеи — в [Issues](https://github.com/DanielLetto2020/LiteEditorAI/issues).
